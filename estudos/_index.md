# Estudos

Aprendizado formal: livros, cursos, artigos, palestras. Tudo que é estudo deliberado e que não cabe melhor em outro domínio.

> Nota: assuntos técnicos vão para `tecnico/`. Teológicos para `espiritual/`. Aqui mora o resto — filosofia, história, ciência, pedagogia, qualquer área que eu esteja estudando.

- [[entities/]] — autores, escolas de pensamento, instituições
- [[sources/]] — livros, cursos, artigos, palestras consumidos
- [[notes/]] — anotações de estudo, exercícios
- [[synthesis/]] — minha síntese consolidada de áreas de estudo
