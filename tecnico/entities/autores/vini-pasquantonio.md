# Viní Pasquantônio (pasquadev)

Engenheiro de software brasileiro. Cria conteúdo no YouTube pelo canal **pasquadev**, focado em programação funcional, DDD, Event Sourcing e arquitetura de software. Tem experiência de trabalho na Alemanha em sistema de apostas esportivas usando Scala — contexto onde viveu na prática os benefícios de imutabilidade e programação funcional em sistemas de alta escala.

## Posições centrais

- Programação funcional é a resposta estrutural para bugs de estado mutável compartilhado
- Event Sourcing + DDD + funcional é uma combinação que faz o código "envelhecer como vinho"
- Engenheiros devem ler papers e código de referências, não apenas escrever mais código
- "Código está barato. Código ruim é extremamente caro."
