# Docker

*A ser preenchido quando houver fonte dedicada.*

Plataforma de containerização — empacota aplicação e dependências em uma unidade portável e isolada. Passo padrão de infraestrutura antes da orquestração via [[tecnico/entities/kubernetes|Kubernetes]]: containerizar sozinho não basta em produção com múltiplas réplicas, é a base sobre a qual o orquestrador atua.
