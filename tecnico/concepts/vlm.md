# VLM (Vision-Language Model)

*A ser preenchido quando houver fonte/contexto.*

Modelos de linguagem que processam tanto texto quanto imagens como entrada. São a fundação do [[vrag]] — substituem o pipeline OCR → texto → LLM por percepção visual direta.

Exemplos conhecidos: Qwen2.5-VL, Qwen3-VL (Alibaba), GPT-4o (OpenAI), Gemini (Google).
