# VimRAG (Visual Memory RAG)

*A ser preenchido quando houver fonte/contexto.*

Evolução do [[vrag]] proposta pela [[alibaba-nlp]]. Usa um **Grafo de Memória Multimodal** para lidar com contextos visuais massivos — organiza o raciocínio como um grafo dirigido em vez de uma lista linear de chunks recuperados. Indicado para casos onde a escala visual inviabiliza a abordagem iterativa simples do VRAG.
